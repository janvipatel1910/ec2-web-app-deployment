# 🚀 EC2 Web App Deployment

## 📌 Project Overview
Deployed a static web application on AWS EC2 instance using Apache web server.

## 🛠️ Technologies Used
- AWS EC2
- Linux (Amazon Linux)
- Apache (httpd)
- HTML

## ⚙️ Steps Performed
1. Launched EC2 instance
2. Configured security group (HTTP - Port 80)
3. Installed Apache web server
4. Created index.html
5. Deployed website on EC2
